Conceito de Div's

Definição:

Uma div esta relacionado a criar "divisões" na página para que possamos posicionar os elementos na página HTML. Existem tags semanticas para as divs, por exemplo a tag header, utilizadas para dividir o conteúdo da página.

Estamos começando a entender o conceito de div.

Vamos criar uma outra div com outras formatações.

Vantagens de DIVS

  • Maior praticidade ao distribuir elementos na página
  • Maior praticidade ao estilizar elementos
  • É divertido

Desvantagens de DIVS

  • Um pouco complexo o entendimento
  • É necessário esforço para aprender
  • Só depende de você entender o funcionamento
